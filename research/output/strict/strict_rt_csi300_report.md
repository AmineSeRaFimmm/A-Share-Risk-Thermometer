## 4. 最终推荐买卖规则（全样本 + IS/OOS 均衡最优）

### 操作规则（可直接落地研究）

| 步骤 | 规则 |
|------|------|
| **买入点 BUY** | 当日收盘同时满足：① **60 ≤ 风险温度 < 80**；② 沪深300 相对 60 日高点回撤 **≤ −5%**（`close / rolling_max_60 − 1 ≤ −0.05`） |
| **卖出点 SELL** | **固定持有 5 个交易日**后，在第 5 日收盘对应的下一交易日开盘卖出（即持有约 5–6 个交易日跨度） |
| **执行时点** | 信号日 T 收盘确认 → **T+1 开盘**买入/卖出（回测已含双边手续费+滑点） |
| **仓位** | 同一时间最多 1 笔；新信号若仍在持仓中则忽略（不重叠加仓） |
| **不做** | 不裸买 RT≥90 极端区；不取消 60 日回撤确认 |

## 5. 使用注意（严格声明）

1. 这是历史回测，不是投资建议；未来分布可能漂移。
2. 高温区间样本天然少于中温区间；强制大样本会倾向中高温带而非极端恐慌裸买。
3. 若存在 `zero_trade_year`，说明某些年份无信号，不可当作全年永续 alpha。
4. 生产前建议纸面跟踪至少一个完整风险周期，并核对数据生成时点 vs 可交易时点。
